# Salvage Sack

A RuneLite plugin that tracks salvage loot from the OSRS Sailing skill.

## Features

- **Track by Shipwreck Type**: Separate tracking for Small, Medium, and Large shipwrecks
- **Detailed Statistics**: 
  - Current drop rate (updated after each loot)
  - Expected drop rate from wiki
  - Total number of drops per item
- **Visual Display**: 
  - Item icons from the wiki
  - Drop counts next to each icon
  - Organized by shipwreck type
- **Persistent Data**: All tracking data is saved between sessions

## How to Use

1. Install the plugin in RuneLite
2. Enable "Salvage Sack" in the plugin panel
3. Go sailing and salvage shipwrecks!
4. View your loot statistics in the Salvage Sack panel (accessible from the sidebar)

## Data Display

The plugin shows:
- Shipwreck type (Small/Medium/Large)
- Total number of loots from each shipwreck type
- For each item:
  - Item icon
  - Item name
  - Number of times dropped
  - Current drop rate vs Expected drop rate

All data persists between RuneLite sessions.
